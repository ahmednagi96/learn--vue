. الـ created (مرحلة الولادة - Logic Only)

في هذه المرحلة، يكون المكون قد "وُلد" برمجياً. الـ Vue قام بتجهيز البيانات (data) والعمليات (methods) والـ computed.

    ماذا يمكنك فعله؟ يمكنك الوصول لـ this.name أو استدعاء دالة. هي أنسب مكان لعمل API Call (جلب البيانات من السيرفر).

    ما الذي لا يمكنك فعله؟ لا يمكنك لمس الـ HTML (الـ DOM) لأن المكون لم يُرسم بعد على الشاشة.

    تشبيه: الممثل حفظ دوره (البيانات) وجاهز في الكواليس، لكنه لم يخرج على المسرح بعد.

2. الـ mounted (مرحلة الظهور - Logic + UI)

هنا يكون المكون قد تم "تركيبه" (Mounted) فعلياً على شاشة المتصفح. الـ HTML أصبح موجوداً ويمكنك رؤيته.

    ماذا يمكنك فعله؟ أي شيء يتطلب التعامل مع الـ HTML مباشرة (مثل استخدام مكتبة رسوم بيانية Chart.js أو التركيز على حقل إدخال input.focus()).

    تشبيه: الممثل الآن واقف على المسرح والجمهور يراه.

3. الـ unmounted (مرحلة الوفاة - Cleanup)

هذه المرحلة تحدث عندما يختفي المكون من الشاشة (مثلاً انتقلت لصفحة أخرى أو استخدمت v-if="false").

    ماذا يمكنك فعله؟ التنظيف (Cleanup). إذا كنت قد بدأت "عداد وقت" (setInterval) أو مستمعاً لحدث عالمي (window.addEventListener)، يجب أن تغلقه هنا حتى لا يستهلك ذاكرة الجهاز والمتصفح مغلق (Memory Leak).

    تشبيه: الممثل غادر المسرح، ويقوم الآن بخلع ملابس الشخصية وإطفاء الأنوار خلفه.
